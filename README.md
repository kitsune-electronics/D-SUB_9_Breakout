# KE-DB09-01 D-SUB 9 Breakout

D-SUB 9ピンの信号をバナナジャックへ引き出すブレイクアウトボックスです。  
電子機器の測定・デバッグ用途を想定しています。

---

## 特徴

- D-SUB 9ピンをストレート結線で中継
- 各ピンの信号をバナナジャックへ引き出し可能
- オス／メス両コネクタを搭載し柔軟な接続に対応
- 測定・デバッグ用途に最適

## 仕様

- コネクタ  
  - D-SUB 9ピン オス ×1  
  - D-SUB 9ピン メス ×1  
- バナナジャック：Φ4 ×9
- 最大定格電圧：30 V（参考値）
- 最大定格電流：2 A（1ピンあたり）
- サイズ：W70 × D100 × H35 mm

※詳細は製品仕様書を参照してください

---

## 使用方法

1. D-SUBケーブルを本製品に接続し、ロックスクリューを締めます  
2. 接続先機器にケーブルを接続します  
3. 引き出したいピンに対応するバナナジャックへ接続します  
4. 測定器などへ接続して使用します
